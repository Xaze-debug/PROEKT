Описание программы "Товарный блокнот"

Программа "Товарный блокнот" - это простое и удобное приложение для учета продуктов и разных товаров(в частности для составления списка продуктов, как для обновления продуктовой корзины так и иметь общую информацию о продуктах и товарах в доме). Она позволяет пользователю добавлять новые товары, просматривать список, отмечать приобретенные товары галочкой, а также управлять списком.

Основные функции:
Альфа версия

- Добавление новых товаров с указанием их названий

- Просмотр текущего списка товаров

- Удаление товаров из списка

- сохранение данных только на 1 сессию входа

Возможно в будущем:
- Отметка товара как приобретенного (вычеркивание галочкой)

- Сохранение и загрузка данных между сессиями


Цели программы:

Создать интуитивно понятный интерфейс для быстрого и удобного учета товаров, при этом обеспечить стабильность и расширяемость функционала.

Используемые технологии:

- Язык программирования
C++ и библиотека Qt для визуализации

Возможно
- Стандартная библиотека (STL) для хранения данных и управления ими
